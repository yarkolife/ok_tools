  БЫСТРЫЙ СТАРТ: Подключение NAS хранилища для видеофайлов
═══════════════════════════════════════════════════════════════════

NAS: \\192.168.188.1\sendedaten (SMB/CIFS)

█ ШАГ 1: Настрой credentials

Отредактируй: scripts/mount_nas.sh

  NAS_USERNAME="твой_логин"     ← ЗАМЕНИ!
  NAS_PASSWORD="твой_пароль"    ← ЗАМЕНИ!

█ ШАГ 2: Монтируй NAS

  ./scripts/mount_nas.sh

Проверь:
  ls -la /Volumes/sendedaten

█ ШАГ 3: Создай структуру папок на NAS

  /Volumes/sendedaten/
  ├── archive/          ← Архив видео
  └── playout/          ← Файлы для эфира

█ ШАГ 4: Обнови docker-compose.yml (УЖЕ ГОТОВО!)

  volumes:
    - /Volumes/sendedaten:/mnt/nas:ro  

Для записи измени :ro → :rw

█ ШАГ 5: Обнови docker.cfg (УЖЕ ГОТОВО!)

  [media]
  archive_path = /mnt/nas/archive/
  playout_path = /mnt/nas/playout/

Измени пути если структура другая!

█ ШАГ 6: Перезапусти Docker

  docker-compose down
  docker-compose up -d

█ ШАГ 7: Проверь доступ

  ./scripts/test_nas_access.sh

█ ШАГ 8: Настрой Storage Locations

  1. Открой: http://localhost:8000/admin/
  2. Логин: admin@example.com / admin123
  3. Перейди: Media Files → Storage locations
  4. Создай:
     
     Name: NAS Archive
     Type: ARCHIVE
     Path: /mnt/nas/archive/
     
     Name: NAS Playout  
     Type: PLAYOUT
     Path: /mnt/nas/playout/

█ ШАГ 9: Сканируй видео

  docker-compose exec web python manage.py scan_video_storage

═══════════════════════════════════════════════════════════════════

✅ ГОТОВО! Видеофайлы найдены и доступны в админке.
